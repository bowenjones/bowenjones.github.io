Binomial Distribution

Notated as: \[X~Binom(n, p) \] Where random variable X follows the Binomial Distribution with n trials of probability p. To determine expected value E(X):

\[E(X) = n * p\] So, to find the expected value E(X) of X~Binom(10, .5):

(10*.5)
## [1] 5

Variance is computed using \[Var(X)= E(X) * (1-p)\]

Determining the probability of a specific outcome x

Notated as \(Pr(X=x)\), where x is the desired number of successes.

Simulation Method

A common method of determing \(Pr(X=x)\) is by running simulations and averaging the results. We use the rbinom function as follows: \[rbinom(n, size, p)\] Note that here, n is desired number of simulations, size is the number of trials and p is the probability of success. To determine the probability of x=5 successes given X~Binom(10, .5),

mean(rbinom(10000, 10, .5)==5)
## [1] 0.2529

This returns the fraction of outcomes with 5 successes.

Exact Method

Alternatively, one could use the dbinom function to find the exact value as follows, dbinom(n, size, p), where the first argument specifies the desired number of successes. So,

dbinom(5, 10, .5)
## [1] 0.2460938

mean(rbinom(10000, 10, .5)<=4) or, pbinom(4, 10, .5)

fair <- dbinom(13, 20, 0.50) bias <- dbinom(13, 20, 0.80) fair / (fair + bias)

Geometric Distribution

Waiting for first success (heads of flips, in this case; “machine malfunctions” for application)

flips <- rbinom(100, 1, .1)

Using the which function,

which(flips == 1)[1] 
## [1] 7

For simulation method use the replicate function, yielding a list of the first appearance of a success for each of 10 trials

replicate(10, which(rbinom(100, 1, .1)==1)[1])
##  [1]  4 23 47  7 13 11  1  1 17 19

To determine expected value, run mean

mean(sum(replicate(10, which(rbinom(100, 1, .1)==1)[1])))
## [1] 55
